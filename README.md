# Data-Cleaning
Student Data Cleaning & Preprocessing Pipeline
Developed an end-to-end Python pipeline to transform highly inconsistent student datasets into structured, analysis-ready formats. Leveraging Pandas, I resolved complex data quality issues, including missing values, duplicate records, and inconsistent string or datetime formatting.

The system efficiently parsed and flattened nested JSON structures to extract granular insights from multi-level data, such as student skills and device information. To ensure data integrity, I implemented rigorous validation using Regular Expressions (Regex) for contact details like emails and phone numbers. Additionally, raw address data was decomposed into structured attributes (city, district, postal code) to enhance downstream geographical analytics.

By standardizing categorical and numeric data, the pipeline delivered a final cleaned CSV dataset optimized for high-level reporting and predictive modeling. This project significantly reduced manual data preparation time and improved the accuracy of educational insights.
